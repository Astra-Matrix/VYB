# 01_VISION — VYB Vision

VYB is designed to become the next-generation game and worldbuilding creation environment: a fusion of editor UX, engine architecture, asset pipelines, cinematic tooling, and AI-assisted development.

## What “next-generation” means
- A WebGPU-first rendering architecture with future native GPU integration paths.
- Studio-grade workflows: project system, asset registry, scene hierarchy, and plugin extensibility.
- Clear interfaces and honest compatibility reporting.
- Documentation and developer experience as first-class product features.

## Current implementation status (scaffold)
This scaffold includes:
- Tauri + React + TypeScript studio shell
- typed ECS/scene scaffolding
- import detection architecture and readable report generator
- rendering abstraction with a placeholder viewport renderer

Full engine/runtime execution, advanced translators, and node editors are planned for later phases.
